ЗВ'ЯЗАТИСЯ З НАМИ

NAVIZONE APP

Ми розробили мобільний додаток NAVIZONE APP для платформи Android і створили API сервер для даного мобільного додатку. NAVIZONE APP дозволяє визначати місце розташування навіть без GPS. Замість супутникових сигналів відбувається тріангуляція сигналів WiFi та веж мобільного зв’язку. Система працює на базі спільних даних.
